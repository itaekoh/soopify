# Soopify + DocNamu 통합 프로젝트 계획서

## 📋 프로젝트 개요

### 목표
- **soopify.com**과 **docnamu.com**을 단일 플랫폼으로 통합
- 전체 디자인 시스템 재설계
- 나무의사 커뮤니티 + 수목 관리 서비스의 통합 플랫폼 구축

### 통합 도메인
- **메인**: `soopify.com`
- **리디렉션**: `docnamu.com` → `soopify.com`

---

## 🎯 서비스 포지셔닝

### 새로운 Soopify
**"생활권 수목 관리 × 나무의사 전문가 커뮤니티"**

#### 핵심 가치
1. **전문가 네트워크**: 나무의사 1,737명의 지식 공유 플랫폼
2. **실무 도구**: 나무병원/보호수 현황 대시보드
3. **비즈니스 서비스**: 수목 조사, 컨설팅, 시스템 개발

## 🗄️ 데이터베이스 설계

### 테이블 구조 (접두어 규칙)

#### 기존 유지 (변경 없음)
```sql
-- SoopNote 테이블 (건드리지 않음!)
sn_*

-- 기존 Soopify 테이블
posts              -- 공지사항
contact_inquiries  -- 문의
```

#### 새로 추가 (docnamu 통합)
```sql
-- 커뮤니티 (접두어: community_)
community_discussions          -- 토론 게시판
community_discussion_comments  -- 토론 댓글
community_discussion_votes     -- 토론 투표
community_discussion_tags      -- 토론 태그
community_qna                  -- Q&A 게시판
community_qna_answers          -- Q&A 답변
community_qna_votes            -- Q&A 투표

-- 전국 현황 (접두어: national_)
national_tree_hospitals        -- 나무병원 정보
national_protected_trees       -- 보호수 정보
national_statistics           -- 통계 데이터

-- 정보센터 (접두어: info_)
info_education                -- 교육 정보
info_news                     -- 뉴스
info_events                   -- 이벤트
info_regulations              -- 법령/제도

-- 공통 (접두어: 없음)
categories                    -- 카테고리
tags                          -- 태그
profiles                      -- 사용자 프로필 확장
```

### 마이그레이션 전략
1. **테이블 생성**: 접두어 규칙에 따라 순차 생성
2. **데이터 이전**: docnamu → soopify (필요시)
3. **인덱스 최적화**: 검색/정렬 성능 최적화
4. **RLS 정책**: 보안 및 권한 관리

## 📱 페이지별 설계 (시안 제작 필요)

### 1. 홈페이지 (/)
**목적**: 통합 플랫폼 소개 및 주요 서비스 안내

**섹션**:
- Hero: 통합 플랫폼 메시지
- Services: 3가지 비즈니스 서비스
- Community: 커뮤니티 미리보기
- Dashboard: 전국 현황 요약
- CTA: 회원가입/문의

### 2. 커뮤니티 (/community)
**하위 페이지**:
- `/community/discussion` - 토론 게시판
- `/community/qna` - Q&A 게시판
- `/community/experts` - 전문가 목록

**주요 기능**:
- StackOverflow 스타일 UI
- 투표, 댓글, 태그, 검색
- 답변 채택 시스템

### 3. 전국 현황 (/dashboard)
**하위 페이지**:
- `/dashboard/hospitals` - 나무병원 현황
- `/dashboard/trees` - 보호수 현황
- `/dashboard/statistics` - 통계

**주요 기능**:
- 지도 시각화
- 차트 및 그래프
- 지역별 필터링

### 4. 정보센터 (/info)
**하위 페이지**:
- `/info/notices` - 공지사항
- `/info/education` - 교육 정보
- `/info/news` - 뉴스
- `/info/regulations` - 법령/제도

### 5. 관리자 (/admin)
**기존 확장**:
- 커뮤니티 관리
- 대시보드 데이터 관리
- 통계 및 분석
